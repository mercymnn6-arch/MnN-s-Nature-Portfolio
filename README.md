# Mercy's Photography Portfolio
#### Wildlife & nature photography showcase, Jan 2026
#### By **Mercy**

## Description
Photography portfolio featuring favorite Kenyan wildlife photos from Nairobi National Park, Amboseli, and Lake Naivasha. Built with semantic HTML5, custom CSS styling, and responsive design principles.

## Live Site
[TBD: mercy.github.io/mercy-photography]

## Setup/Installation Requirements
1. Clone this repository
2. Open `index.html` in any modern web browser
3. No additional setup required

## Technologies Used
- HTML5 semantic elements (header, section, aside, article)
- CSS3 (Flexbox layout, float positioning, box model)
- Responsive images with proper alt text
- Form validation (HTML5)

## Known Bugs
